Point of Sales Discount Control

This module add the missing piece of POS restriction features

User Interface Changes

  • Natively, Odoo allows to set the restriction of price modifications to managers

  • This module fills the missing piece by allowing the user to set discount restrictions to managers.

Configurable POS by POS

  • The restriction can be turned on / off for each POS via its configure page.
