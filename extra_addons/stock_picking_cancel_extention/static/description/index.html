Picking(Delivery Orders/Receipts) Cancel/Reverse

This module is developed to do Picking cancel which is done state.

Key Features

  • Give access rights to Cancel Delivery Order/Receipt.
  • Cancel Delivery Order/Receipt which are done state.
  • Also cancel Delivery Order/Receipt for By Unique Serial Number and By Lots products.
  • Reset Picking to Draft state
  • Get update on Product On hand quantity

Configuration

Cancel button is available on Done state.

Once you click on the "Cancel" button, if there is Delivery Order than Quantity is Increase and if there is Receipts qty is decrease
Once you cancel picking set it to the Draft state.
Set Draft state of the Cancelled picking.
