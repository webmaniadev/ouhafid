Au niveau de la vue formulaire "Journal":

  • Ajout d’une case à cocher "Utiliser pour les paiements sur bon de commande".
  • Ce jour doit être positionné plus bas que le journal de ventes principale.
  • Pendant la création de journal, il est nécessaire de cocher le champs "Utiliser pour les paiements sur bon de commande".

Au niveau de la vue formulaire "Devis":

  • Ajout d’une case à cocher "Paiement sur Bon de commande".
  • Si cette case à cocher est activée:
    • Apparition d’un onglet "Règlements" pour le suivi des paiements.
    • Apparition de "l’état de la vente" & "Montant dû".

Au niveau de la vue liste des "Devis":

  • Ajout de la colonne "ventes au détails" indiquant les Bon de commande Valorisés.
  • Ajout d’un filtre "Vente au détail" pour afficher uniquement les Bons de commande valorisés.

Ajout d’un rapport "Bon de commande Valorisé" contenant:

  • Quantité commandée.
  • Prix unitaire.
  • Taxes.
  • Montant en lettre.
  • Rectangle pour le visa du vendeur.
  • Rectangle pour l'accusé de réception du destinataire.

Ajout d’un rapport "Bon de livraison Valorisé" contenant:

  • Quantité commandée.
  • Prix unitaire.
  • Taxes.
  • Montant en lettre.
  • Rectangle pour le visa du vendeur.
  • Rectangle pour l'accusé de réception du destinataire.

Implementations


  • Journal
  • Quotation form
  • Quotations lists
  • Sale order
  • Delivery slip

In the "Journal" view form:

  • Add a "Use for sale order payments" checkbox.
  • This journal should be positioned lower than the principal sales journal.
  • During journal creation, it is necessary to check the field "Use for sale order payments".

In the view form "Quotation":

  • Add a "Sale order payments" checkbox.
  • If this checkbox is enabled:
    • Appearance of a "Regulations" tab for monitoring payments.
    • Appearance of "Sale status" & "Amount due".

In the view list "Quotations":

  • Add a "Retail sales" column indicating the valued sales orders.
  • Add a "Retail" filter to display only valued sales orders.

Add a "Valued Sale Order" report containing:

  • Ordered quantity.
  • Unit price.
  • Taxes.
  • Amount in letter.
  • Rectangle for vendor visa.
  • Rectangle for the recipient acknowledgement.

Add a "Valued Delivery Sale" report containing:

  • Ordered quantity.
  • Unit price.
  • Taxes.
  • Amount in letter.
  • Rectangle for vendor visa.
  • Rectangle for the recipient acknowledgement.
